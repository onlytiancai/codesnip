# 第 2 章 一个神经元 = 一个函数

> **回链**：本章用到 0.1 的"函数"概念（[第 0 章 0.1 节](#01-函数丢进一个数吐出一个数)）。

## 2.1 神经元公式拆解

把上一章的"小精灵"翻译成数学：

$$
y = \sigma(w \cdot x + b)
$$

**4 个参数**：
- `x`：输入（数字或向量）
- `w`：权重（与 x 形状相同，决定每个 x 的重要性）
- `b`：偏置（"门槛"——b 越大，神经元越难激活）
- `σ`：激活函数（把任意实数压到 (0, 1)）

## 2.2 权重 w、偏置 b 是什么

**权重 w**（weight）：
- "重要性"——w 越大，对应输入越重要
- w > 0：兴奋（这个特征让神经元更可能激活）
- w < 0：抑制（这个特征让神经元更不可能激活）
- w = 0：这个输入被"忽略"

**偏置 b**（bias）：
- "门槛"——b 越大，神经元越难激活
- b = 0：门槛是 0
- b = -10：门槛很低，神经元容易激活
- b = 10：门槛很高，神经元几乎不激活

**实际例子**：判断"能不能出去玩"
- x1 = 天气好坏（0=差，1=好）
- x2 = 温度舒适度（0~1）
- x3 = 是否有作业（0=无，1=有）
- w = [0.5, 0.3, -0.8]（天气重要，作业是减分项）
- b = -0.2（门槛很低，大多数情况都出去）
- 输出 y = σ(0.5·x1 + 0.3·x2 - 0.8·x3 - 0.2)
- y 越接近 1 → 越倾向"出去"；越接近 0 → 越倾向"宅家"

## 2.3 例：出门决策——3 个输入变成 1 个数字

**场景**：你今天要不要出去玩？3 个因素：温度、湿度、是不是周末。

| 输入 | 含义 | 数值 |
|---|---|---|
| x1 | 温度（0=冷，1=热） | 0.8 |
| x2 | 湿度（0=干，1=湿） | 0.3 |
| x3 | 是不是周末（0=工作日，1=周末） | 1.0 |

权重（学到的，假定）：
- w1 = 0.5（温度很重要）
- w2 = -0.3（湿度大是减分）
- w3 = 0.4（周末加分）
- b = -0.2

**计算**：
1. 加权求和：sum = 0.5·0.8 + (-0.3)·0.3 + 0.4·1.0 + (-0.2) = 0.4 - 0.09 + 0.4 - 0.2 = 0.51
2. 激活：y = σ(0.51) ≈ 0.625
3. 解读：y = 0.625，倾向"出门"（>0.5 就算支持出门）

::: formula
$$
\sigma(z) = \frac{1}{1 + e^{-z}}
$$
:::
> 上面是 sigmoid 公式——把任意实数 z 压到 (0, 1) 区间。第 5 章会详细讲。

::: quiz q2-1 single
神经元公式 $\sigma(w \cdot x + b)$ 中，4 个部分（$x, w, b, \sigma$）哪个最像"门槛"？
- A: $x$（输入）
- B: $w$（权重）
- C: $b$（偏置）
- D: $\sigma$（激活函数）

answer: C

> 偏置 b 控制"门槛"——b 越大，神经元越难激活。
:::

::: quiz q2-2 single
神经元输出 $y = \sigma(w \cdot x + b)$ 的范围是？
- A: 任意实数（-∞, +∞）
- B: [0, 1]
- C: (0, 1)（开区间）
- D: {-1, +1}

answer: C

> σ 把任意实数压到 (0, 1) 开区间（不会真到 0 或 1，但可以无限接近）。这是 sigmoid 函数的特性。
:::

::: quiz q2-3 short placeholder="用 1 句话向朋友解释什么是权重"
请用**一句话**向你的朋友解释什么是"权重"。

> 参考答案（不唯一）：
> - "权重就是这个特征的重要性分数——分数高，它对最终结果的影响就大；分数是负的，就是扣分项"
> - "权重是'偏心'——权重大的特征，神经元更'在意'它"
> - "权重是音量旋钮——权重越大，特征的声音越大"
:::
