## 0.1 函数：丢进一个数，吐出一个数

> **类比：自动售货机**。投 1 个硬币进去（输入），按一个按钮（公式 f），掉出一瓶饮料（输出）。

::: chart caption="函数就是机器：丢进一个数，吐出另一个数"
![函数机器](assets/images/ch00_function_machine.png)
:::

**形式**：`y = f(x)`，读作"y 等于 f 作用于 x"。

**例子**：`f(x) = 2x + 1`（加倍再加 1）

| 输入 x | 计算 | 输出 y |
|---|---|---|
| 0 | 2·0 + 1 | 1 |
| 1 | 2·1 + 1 | 3 |
| 3 | 2·3 + 1 | 7 |

**关键概念**：
- **输入域**：所有可以丢进去的 x（这里是任意数字）
- **输出域**：所有会吐出来的 y
- **自变量 vs 因变量**：x 是自变量（自己变），y 是因变量（跟着 x 变）

::: quiz q0-1-1 single
下面哪个**不是**一个合法的函数？
- A: $f(x) = 3x + 2$
- B: $f(x) = x^2$（平方）
- C: $f(x) = $ "如果 x 是偶数返回 0，否则返回 1"（"魔改"函数）
- D: $f(x) = $ "x 是数字，返回 x+1；否则报错"

answer: D

> 函数的定义要求每个输入都有**唯一**输出。选项 D 里，对非数字的输入没有规定输出（报错不是输出），所以不构成"完整的函数"。但实际上，编程中常常只定义在"合法输入"上的函数，这叫**偏函数**。
:::

::: quiz q0-1-2 single
$f(x) = 3x - 5$，求 $f(4)$。
- A: 7
- B: 12
- C: -7
- D: 5

answer: A

> 代入：$3 \times 4 - 5 = 12 - 5 = 7$。注意运算顺序：先乘法后减法。
:::

## 0.2 坐标系：把数字画在地图上

> **类比：电影院座位**。第几排（横轴）、第几列（纵轴）—— 两个数字就定位一个座位。

::: chart caption="坐标系：横轴 x、纵轴 y，两个数字定一个点"
![坐标系](assets/images/ch00_coordinate_grid.png)
:::

**笛卡尔坐标系**：
- **横轴（x 轴）**：向右为正
- **纵轴（y 轴）**：向上为正
- **原点 (0, 0)**：两轴交点
- **点 (a, b)**：横坐标 a + 纵坐标 b
- **4 个象限**：右上（+,+）、左上（-,+）、左下（-,-）、右下（+,-）

**为什么要坐标系？**
因为函数 `f(x) = y` 是"输入 → 输出"的关系。把所有 (x, y) 画在坐标系里，就能看到函数的"形状"。

::: quiz q0-2-1 single
点 $(3, -2)$ 在哪个象限？
- A: 第一象限（右上）
- B: 第二象限（左上）
- C: 第三象限（左下）
- D: 第四象限（右下）

answer: D

> x=3 > 0（向右），y=-2 < 0（向下），所以是右下 → 第四象限。
:::

::: quiz q0-2-2 short placeholder="描述你家的位置（用 x 和 y 坐标）"
请用**一组坐标**描述你家的位置（不需要真实地图，随便编一个坐标系也行）。

> 参考答案：
> - "我家在 (2, 3) 这个位置——以学校为原点，向东走 2 公里，向北走 3 公里"
> - "我家在 (-1, 4)，在学校西边 1 公里、北边 4 公里"
:::

## 0.3 斜率：函数在某点"有多陡"

> **类比：汽车速度表**。速度表显示的不是"位置"而是"位置的变化快慢"。导数 = "在某个瞬间你跑多快"。

::: chart caption="f(x) = x² 在 x=1.5 处的切线（斜率=3）"
![斜率](assets/images/ch00_slope_tangent.png)
:::

**什么是斜率**：
- 斜率 = "y 变化多少" / "x 变化多少"
- 直线 `y = mx + b` 的 m 就是斜率
- 斜率为正：函数向右上方走；为负：向右下方

**例**：
- `y = 2x + 1`：斜率 m=2，每向右一步，y 上升 2 步
- `y = -3x + 5`：斜率 m=-3，每向右一步，y 下降 3 步

**函数在某点的斜率**（=导数）：
- `f(x) = x²`，斜率 `f'(x) = 2x`
- `f(3) = 9`，`f'(3) = 6`（在 x=3 处，函数上升速度是 6）

::: quiz q0-3-1 single
直线 $y = 2x - 1$ 的斜率是多少？
- A: 2
- B: -1
- C: 1
- D: 0.5

answer: A

> 直线 $y = mx + b$ 的 m 就是斜率。这里 m=2。
:::

::: quiz q0-3-2 short placeholder="用 1 句话解释斜率"
请用**一句话**向你妈妈解释什么是"斜率"。

> 参考答案（不唯一）：
> - "斜率就是'这条路有多陡'——斜率大，路陡；斜率小，路平；负斜率就是反着走"
> - "斜率是函数在某点的'脾气'——脾气急（斜率大），稍微一推就涨很多；脾气缓（斜率小），推很多才涨一点"
:::

## 0.4 向量：一列数字描述一个对象

> **类比：个人档案表**。身高、体重、年龄、成绩——4 个数字就能描述一个人。

::: chart caption="向量：一列有序的数字"
![向量](assets/images/ch00_vector_personal.png)
:::

**向量的定义**：
- **一组有序的数字**
- 用方括号 `[a, b, c]` 表示
- **几维** = 几个数字

**例子**：
- `[165, 50, 12, 92]`：4 维向量，描述"小明"的身高/体重/年龄/成绩
- `[3, 5]`：2 维向量，可以画在坐标系里
- `[28, 28, 1]`：手写数字图片的形状（28×28 像素、1 通道）

**为什么神经网络需要向量？**
- 神经网络的"输入"就是一个向量
- 一张图片 = 784 维向量（28×28 像素展平）
- 一个学生的档案 = 4 维向量

::: quiz q0-4-1 single
`[3, 5, 7, 9]` 是几维向量？
- A: 3
- B: 4
- C: 5
- D: 9

answer: B

> 数一下有多少个数字：3, 5, 7, 9 —— 4 个，所以是 4 维。
:::

::: quiz q0-4-2 short placeholder="写一个 5 维向量描述你自己"
请用**5 个数字**写一个描述你自己的向量。

> 参考答案（不唯一）：
> - `[12, 165, 50, 92, 7]`：年龄 12、身高 165cm、体重 50kg、数学成绩 92、最喜欢的数字 7
> - `[165, 50, 165, 50, 165]`：单纯的身高体重循环
:::

## 0.5 矩阵与矩阵乘法：表格的魔法

> **类比 1：Excel 表格**。每行一个学生，每列一个科目。
> **类比 2："批量算账"**。一张发票 × 一个税率表 = 实付金额表。

::: chart caption="矩阵乘法：(2×2) × (2×1) = (2×1)"
![矩阵乘法](assets/images/ch00_matrix_multiplication.png)
:::

**矩阵的形状**：
- `(行数 × 列数)`：2 行 3 列 = `(2, 3)`
- **每行**是一个向量，**每列**也是一个向量

**矩阵乘法规则**：
- **形状匹配**：左矩阵的**列数** = 右矩阵的**行数**
- 结果矩阵的形状：(左行数, 右列数)
- **每个格子 = 左矩阵对应行 × 右矩阵对应列**（点积 = 对应位置相乘后求和）

**例子**：(2×2) × (2×1) = (2×1)
$$
\begin{bmatrix} 1 & 2 \\ 3 & 4 \end{bmatrix} \times \begin{bmatrix} 5 \\ 6 \end{bmatrix} = \begin{bmatrix} 1·5+2·6 \\ 3·5+4·6 \end{bmatrix} = \begin{bmatrix} 17 \\ 39 \end{bmatrix}
$$

**为什么神经网络需要矩阵？**
- 批量数据（多个样本）就是矩阵：每行一个样本
- 权重 W1 = 矩阵：每行连到下一层每个神经元
- 整个前向传播 = 一串矩阵乘法 + 激活

::: quiz q0-5-1 single
(2 × 3) 矩阵 × (3 × 4) 矩阵 = ?
- A: (2, 4)
- B: (3, 3)
- C: (2, 3)
- D: (3, 4)

answer: A

> 左矩阵 2 行 3 列，右矩阵 3 行 4 列。结果：(左行数, 右列数) = (2, 4)。
> 验证：左列数 (3) = 右行数 (3) ✓ 形状匹配。
:::

## 0.6 偏导 + 链式法则：嵌套机器的信号

> **类比：飞机 3 个旋钮**（油门、方向舵、升降舵）。每个旋钮对"高度"都有影响，但影响大小不同。**偏导** = 单个旋钮的灵敏度。
>
> **链式法则**：A 影响 B，B 影响 C → A 对 C 的影响 = (A→B 的影响) × (B→C 的影响)。

::: chart caption="链式法则：信号在嵌套函数里反向流动"
![链式法则](assets/images/ch00_chain_rule.png)
:::

**偏导**：`∂f/∂x` 读作"f 对 x 的偏导"——把其他变量都按住，只看 x 变化时 f 怎么变。

**例**：`f(x, y) = 3x + 5y`
- `∂f/∂x = 3`（按住 y，x 增 1，f 增 3）
- `∂f/∂y = 5`

**链式法则**：`f(g(x))` 对 x 的导数 = `f'(g) × g'(x)`

**例**：`f(g) = g²`，`g(x) = 2x + 1`，求 `(f∘g)'(x)`
- `f'(g) = 2g`，`g'(x) = 2`
- `(f∘g)'(x) = 2(2x+1) × 2 = 8x + 4`

**为什么神经网络需要它？**
- 神经网络就是嵌套函数：x → z1 → a1 → z2 → a2 = ŷ
- 反向传播 = 从 ŷ 的损失 L 反推到第一个权重 W1
- 反推每一步都用链式法则

::: quiz q0-6-1 single
$f(x, y) = 4x + 7y$，求 $\partial f / \partial x$。
- A: 4
- B: 7
- C: 11
- D: 0

answer: A

> 按住 y，只让 x 变化。f = 4x + (其他跟 x 无关)，所以偏导 = 4。
:::
