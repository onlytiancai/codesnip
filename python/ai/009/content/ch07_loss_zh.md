# 第 7 章 损失函数：预测错了多少？

## 7.1 三种误差

**损失函数**（loss function）= 衡量"预测值和真值的差距"。

**3 种常见误差**（单样本）：

| 名称 | 公式 | 特点 |
|---|---|---|
| 绝对误差 MAE | $L = |y - \hat{y}|$ | 简单、但 0 点不可微 |
| 平方误差 MSE | $L = (y - \hat{y})^2$ | 处处可微、放大大的错 |
| 交叉熵 CE | $L = -[y \log \hat{y} + (1-y)\log(1-\hat{y})]$ | 二分类标配 |

**二元交叉熵**（Binary Cross-Entropy, BCE）的细节：
- 适用于 2 分类（y=0 或 y=1）
- 真值 y=1 时：L = -log(ŷ)，ŷ 越接近 1，损失越小
- 真值 y=0 时：L = -log(1-ŷ)，ŷ 越接近 0，损失越小

---

## 7.2 二元交叉熵 BCE：直觉 + 公式

**直觉**：BCE 来自"信息论"
- 真值 y=1，预测 ŷ 越接近 1，损失越小
- 真值 y=0，预测 ŷ 越接近 0，损失越小
- **如果预测完全错**（ŷ → 0 但 y=1），损失 → +∞

::: chart caption="BCE 损失：预测越准，损失越小"
![BCE](assets/images/ch07_bce_curve.png)
:::

**举例**：
- y=1, ŷ=0.99：BCE = -log(0.99) ≈ **0.01**（几乎完美）
- y=1, ŷ=0.5：BCE = -log(0.5) ≈ **0.69**（随机猜测）
- y=1, ŷ=0.01：BCE = -log(0.01) ≈ **4.6**（灾难性错误）

---

## 7.3 为什么分类用 BCE 不用 MSE

**为什么分类问题更爱用 BCE？**

| 维度 | MSE | BCE |
|---|---|---|
| 公式 | $(y - \hat{y})^2$ | $-[y\log\hat{y} + (1-y)\log(1-\hat{y})]$ |
| 0 点处梯度 | 小 → 学习慢 | 大 → 学习快 |
| 极端值 | 梯度饱和 | 始终有效 |
| 概率解释 | 无 | 强（KL 散度） |

**梯度消失问题**（MSE + sigmoid 的经典坑）：
- 当 σ 输出接近 0 或 1 时，σ 的导数接近 0
- MSE 梯度经过 σ 导数 → 也接近 0 → 权重几乎不更新 → 学不动
- BCE 梯度经过 σ 导数 → **恰好抵消**了 σ 导数 → 梯度不消失 ✓

**这就是为什么分类问题（输出是概率）几乎都用 BCE 而不用 MSE**。

::: quiz q7-1 single
真值 $y=1$，预测 $\hat{y}=0.01$，BCE 损失约为？
- A: 0.01
- B: 4.6
- C: 0.99
- D: 0

answer: B

> $L = -\log(0.01) = -\log(10^{-2}) = 2 \log 10 \approx 2 \times 2.303 \approx 4.6$。
> 预测 0.01 但真值 1 → **完全错** → 损失爆炸。
:::

::: quiz q7-2 multiple
损失函数必须满足？（多选）
- A: 可微（绝大多数情况）
- B: 越大越好
- C: 反映"预测错的程度"
- D: 输出恒为正

answer: A,C,D

> A ✓：要能算梯度 → 需要可微。
> B ✗：**越小越好**——训练就是减小损失。
> C ✓：核心目的——衡量错多少。
> D ✓：损失必须 ≥ 0（距离的含义）。
:::

::: quiz q7-3 short placeholder="解释 BCE 抗梯度消失"
**问题**：如果把 BCE 换成 MSE（$(y - \hat{y})^2$），用 sigmoid 输出，会出现什么梯度消失问题？

> 参考答案（不唯一）：
> 假设 $y=1$，$\hat{y} = \sigma(z) \approx 0$（z 很小）。
> - MSE 梯度：$\frac{\partial L}{\partial z} = 2(\sigma(z) - y) \cdot \sigma'(z) = 2(0 - 1) \cdot \sigma'(z) \approx -2 \sigma'(z)$。
> - 当 z 很小，$\sigma'(z) \approx 0$ → 梯度 ≈ 0 → 权重几乎不更新。
> 
> 对比 BCE 梯度：$\frac{\partial L}{\partial z} = \sigma(z) - y = 0 - 1 = -1$（**不**含 $\sigma'$）。
> - 即便 z 很小，梯度依然是 -1 → 学习不被"卡住"。
> 
> **BCE 巧妙的"约分"**：把 $\sigma'$ 消掉了，绕开了梯度消失。
:::
