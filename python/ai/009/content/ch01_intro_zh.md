# 第 1 章 什么是神经网络？

> **这一章 0 数学**。我们用"决策小精灵"的故事引入，后面章节再讲公式。

## 1.1 一个神经元：把多个数字加权求和再加激活

想象一个**小精灵**，它要做决定：

- 听到 **3 个数字**（比如：温度 30°、湿度 80%、今天是周末 1）
- 每个数字有一个**重要性权重**（比如：温度很重要，权重 0.5；今天周几不太重要，权重 0.1）
- 小精灵把所有数字"加权求和"
- 然后用一个**开关函数**决定：输出"出门"还是"宅家"

**这就是一个神经元！** 它的本质就 2 步：
1. **加权求和**：`sum = w1·x1 + w2·x2 + w3·x3 + b`
2. **激活**：`y = σ(sum)`（σ 是一个把任意数压成 0~1 的函数）

::: chart caption="一个神经元的结构"
![神经元卡通](assets/images/ch01_neuron_cartoon_zh.png)
:::

**简化版**（想象每个数字是"证据"，权重是"重要性"）：
- x1=30（温度）、w1=0.5 → 贡献 15
- x2=80（湿度）、w2=0.2 → 贡献 16
- x3=1（周末）、w3=0.3 → 贡献 0.3
- b=-20（"门槛"）
- sum = 15 + 16 + 0.3 - 20 = 11.3
- σ(11.3) ≈ 0.9999（强烈支持"出门"）

## 1.2 网络：很多神经元连起来

一个神经元只能做"是/否"的简单决定。**复杂决定需要很多神经元"投票"**。

比如判断"这张图片是猫还是狗"：
- 第 1 排神经元：识别"耳朵形状"（尖的？圆的？）
- 第 2 排神经元：识别"眼睛大小"
- 第 3 排神经元：识别"胡须"
- ...
- 最后 1 个神经元：综合所有信息，输出"猫"或"狗"

**每排神经元 = 一层**。很多层叠起来 = **神经网络**。

::: network
2-4-1 MLP：3 层神经元（输入 2 个 + 隐藏 4 个 + 输出 1 个）
:::

**直觉**：
- 第 1 层学"低级特征"（边缘、颜色）
- 中间层学"中级特征"（耳朵、眼睛）
- 最后一层学"高级概念"（"猫"）

## 1.3 神经网络和大脑的关系

**重要澄清**：神经网络**不是**大脑的复制品。

| 维度 | 大脑神经元 | 人工神经元 |
|---|---|---|
| 数量 | ~860 亿 | 几亿（最大模型） |
| 信号 | 电化学脉冲 | 数字（0~1） |
| 连接 | 动态生长 | 固定数值 |
| 学习 | 突触可塑性 | 梯度下降 |
| 速度 | ~毫秒 | ~纳秒 |

**它们的关系**：
- 大脑神经元启发了人工神经元（1943 年 McCulloch & Pitts）
- 但工作原理**完全不同**
- 人工神经网络是**数学模型**，不是生物模型

::: quiz q1-1 single
一个神经元本质在做哪两件事？
- A: 加权求和 + 激活
- B: 矩阵乘法 + 池化
- C: 卷积 + 全连接
- D: 排序 + 求平均

answer: A

> 神经元 = 加权求和 + 激活函数。这两件事构成了所有神经网络的"基本粒子"。
:::

::: quiz q1-2 multiple
神经元和大脑神经细胞的相同点是？（多选）
- A: 都有多个输入信号
- B: 都能"学习"
- C: 结构一模一样
- D: 都用电信号

answer: A,B

> 相同：都有输入、能学习（A、B ✓）。
> 不同：结构不一样（C ✗）；大脑用电化学脉冲不只是电信号（D ✗）。
:::
