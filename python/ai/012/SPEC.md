# 链式法则导数可视化演示 - 规格说明

## 1. 项目概述

- **项目名称**: Chain Rule Derivative Visualization
- **类型**: 交互式数学教育可视化
- **核心功能**: 通过双图联动展示链式法则 dy/dx = dy/du × du/dx 的几何意义
- **目标用户**: 学习微积分的学生和教师

## 2. 函数定义

### 内层函数 u = g(x) 选项
| 函数 | 表达式 | 导数 du/dx | x 范围 |
|------|--------|-----------|--------|
| 抛物线 | x² | 2x | [-2, 2] |
| 正弦 | sin(x) | cos(x) | [-2π, 2π] |
| 指数 | eˣ | eˣ | [-2, 2] |
| 三次函数 | x³ | 3x² | [-1.5, 1.5] |

### 外层函数 y = f(u) 选项
| 函数 | 表达式 | 导数 dy/du | u 约束 |
|------|--------|-----------|--------|
| 抛物线 | u² | 2u | 无 |
| 正弦 | sin(u) | cos(u) | 无 |
| 指数 | eᵘ | eᵘ | 无 |
| 对数 | ln(u) | 1/u | u > 0 |
| 绝对值 | |u| | u/|u| (u≠0) |

### 函数组合兼容性
- ln(u) + x²: u = x² ≥ 0 ✓
- ln(u) + sin(x): sin(x) 可能为负 ✗ → 限制 x 范围
- |u| + x²: u = x² ≥ 0 ✓
- |u| + sin(x): u 可能为负 ✗ → 限制 x 范围

## 4. 交互规格

### 控件
- **内层函数下拉菜单**: 选择 g(x)
- **外层函数下拉菜单**: 选择 f(u)
- **x 滑块**: 范围根据函数自适应，步长 0.01
- **播放按钮**: 切换播放/暂停状态
- **速度控制**: 0.5x, 1x, 2x

### 动画行为
- 播放时 x 从最小值到最大值循环移动
- 动画周期约 4 秒（1x 速度）
- 暂停时保留当前状态

- 坐标范围: 函数自适应 [-3, 3] 或根据函数调整

## 6. 验收标准

1. ✓ 两个 Canvas 同时显示内层和外层函数图像
2. ✓ 滑块移动时两个图像同步更新
3. ✓ 切线斜率与解析导数值一致
4. ✓ 公式板显示 dy/dx = dy/du × du/dx 的实时计算
5. ✓ 播放动画时 x 自动变化
6. ✓ 函数切换时自动调整 x 范围
7. ✓ ln(u) 等有约束的函数正确处理
