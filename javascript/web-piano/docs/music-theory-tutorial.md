# 小星星和弦编配教程

## 1. 认识音阶和音级

小星星是 C 大调，音阶是：

```
C  D  E  F  G  A  B  C
1  2  3  4  5  6  7  1
```

## 2. 和弦基础

C 大调中最常用的三个和弦：

| 和弦 | 组成音 | 功能 |
|------|--------|------|
| **C (I级)** | C-E-G | 主和弦，稳定 |
| **F (IV级)** | F-A-C | 下属和弦，倾向往下走 |
| **G (V级)** | G-B-D | 属和弦，强烈倾向回到主和弦 |

## 3. 小星星的和弦编配

### 原曲旋律分析

```
小星星原曲：
C C G G | A A G2 |
F F E E | D D C2 |
```

### 编配思路

**规律一：每小节配一个和弦**

常见做法是让和弦跟着小节走，一个小节一个和弦。

**规律二：使用 I-IV-V-I 走向**

最经典的和弦进行是 I → IV → V → I

对于小星星：

```
第一句：C C G G | A A G2 |
        I  I  V  V   (其实这里用 V 更合适，因为 G 是属音，强烈想回 C)

更好的配法：C  C  G  G | G  G  C2 |  ← 每个音配对应和弦
            I  I  V  V    V  V  I
```

**简化版配法：**

```
C  C  G  G | A  A  G2 |   →  C  C  G  G | C  C  G2 |
F  F  E  E | D  D  C2 |   →  F  F  C  C | G  G  C2 |
```

## 4. 左手低音怎么配

低音和弦是根音在下面，和弦音在上面的叠置：

```
C和弦：   E     ← 和弦音（中音）
        C      ← 根音（低音）

写为：C,2  (表示低音 C 持续2拍)
```

## 5. 完整乐理框架

### 调式音阶

C 大调音阶：1(C) 2(D) 3(E) 4(F) 5(G) 6(A) 7(B)

### 三和弦公式

| 级数 | 和弦 | 公式 | 举例 |
|------|------|------|------|
| I | 大三和弦 | 1-3-5 | C-E-G |
| ii | 小三和弦 | 1-b3-5 | D-F-A |
| iii | 小三和弦 | 1-b3-5 | E-G-B |
| IV | 大三和弦 | 1-3-5 | F-A-C |
| V | 大三和弦 | 1-3-5 | G-B-D |
| vi | 小三和弦 | 1-b3-5 | A-C-E |
| vii° | 减三和弦 | 1-b3-b5 | B-D-F |

### 经典和弦进行

```
I - IV - V - I      （万能进行）
I - V - vi - IV     （卡农进行）
I - vi - IV - V     （50年代进行）
```

## 6. 实战技巧

1. **听感优先**：规则是死的，好听是硬的
2. **低音走向**：低音尽量平滑过渡，避免大跳
3. **和弦色彩**：大调曲子多用法和弦，少用减和弦
4. **多次尝试**：同一旋律可以配出多种和弦方案

# 小星星

## 第一版
```
X:1
T:小星星
C:Anonymous
L:1/4
M:4/4
Q:1/4=120
%%staves {(RH) (LH)}
V:RH clef=treble
V:LH clef=bass
K:C
V: RH
CC GG | AA G2 | FF EE | DD C2 |
GG FF | EE D2 | GG FF | EE D2 |
CC GG | AA G2 | FF EE | DD C2 |]
V: LH
C,2 G,2 | E,2 C,2 | F,2 C,2 | G,2 D,2 |
A,2 E,2 | F,2 C,2 | A,2 E,2 | F,2 C,2 |
C,2 G,2 | E,2 C,2 | F,2 C,2 | G,2 D,2 |]
```

说明：
- 右手（RH）：高音谱表，演奏主旋律
- 左手（LH）：低音谱表，演奏低音和弦
- 每小节对应一个和弦进行：C → G → F → C（左手是 C-E-G 和弦的低音）
- 音符命名规则：
    - 小写 c = C4（中央 C）
    - 大写 C = C3（低一个八度）
    - 带逗号如 C, = C2（再低一个八度）

## 七和弦版

```
X:1
T:小星星（七和弦版）
C:Anonymous
L:1/4
M:4/4
Q:1/4=120
%%staves {(RH) (LH)}
V:RH clef=treble
V:LH clef=bass
K:C
V: RH
CCGG | AAG2 | FFEE | DDC2 |
GGFF | EED2 | GGFF | EED2 |
CCGG | AAG2 | FFEE | DDC2 |]
V: LH
C,E,G,B, | C,E,G,B, | G,2 D,G,B, | G,2 D,G,B, |
F,A,C,E, | F,A,C,E, | F,2 C,E,A, | F,2 C,E,A, |
C,E,G,B, | C,E,G,B, | G,2 D,G,B, | G,2 D,G,B, |]
```

变化说明：

|a位置|原版|新版 |
|-|-|-|
|第1、2小节|C大二和弦|Cmaj7 (C-E-G-B) 大七和弦，更柔和|
|第3、4小节|G大二和弦| G7 (G-B-D-F) 属七和弦，更爵士|
|第5、6小节|F大二和弦|Fmaj7 (F-A-C-E) 大七和弦，温暖 |
|第7、8小节 |F+C小和弦|Fmaj7 + Am7 (F-A-C-E + A-C-E-G)|


## 更简单的七和弦版本：

```
X:1
T:小星星（简化七和弦版）
C:Anonymous
L:1/4
M:4/4
Q:1/4=120
%%staves {(RH) (LH)}
V:RH clef=treble
V:LH clef=bass
K:C
V: RH
CCGG | AAG2 | FFEE | DDC2 |
GGFF | EED2 | GGFF | EED2 |
CCGG | AAG2 | FFEE | DDC2 |]
V: LH
C,2 E,2 | G,2 B,2 | G,2 D,2 | G,2 B,2 |
F,2 A,2 | C,2 E,2 | F,2 A,2 | C,2 E,2 |
C,2 E,2 | G,2 B,2 | G,2 D,2 | G,2 B,2 |]
```
这样右手主旋律完全不变，左手低音加上了七度音（E, G, B, A 等），让和声更丰富立体。
