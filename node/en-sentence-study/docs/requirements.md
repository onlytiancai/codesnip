做一个英语句子学习的 web app，目的是锻炼用户的口语表达能力，心里想说的话，能够用英文表达出来
  - 给出一个中文的场景描述，一个该场景下的中文的口语句子，让用户用英文来尝试表达，AI给出点评
  - 给出中文句子后，再给出相关的英文重点单词及解释，目的是让用户能够表达完整句子，而不是考察单词量。
  - 场景，句子由 LLM 生成，历史句子要保存历史，保证句子的多样性，场景覆盖率，历史句子尽量不重复。
  - 提前要有一套场景列表，比如机场，酒店，打车，工作等等，每个场景下有任务列表，用户可以选择场景和任务进行练习，也可以随机，用户可以看到场景，任务学习完成度。
  - 用户给出英文句子后要做出专业的点评，包括单词拼写，语法，表达习惯，是否礼貌，是否地道，是否适合这个场景等
  - 用户可以选择获取提示，AI给出多个候选的英语句子，因为一句中文在英文里可以有多种不同的表达，比如语气不同，侧重不同，有些单词可以用同义词，近义词代替等
  - 学过的句子，有复习功能，复习时自动播放中英文，用户可设置只听英文，还是中应为对照，类似闪卡的设计，用户可以设置学会的句子不再复习
  - 有一些统计功能，比如本日学习句子数，本周学习句子数，有积分系统，勋章系统，每日打卡系统，分享炫耀系统，邀请码系统，日，周排行榜系统
  - 功能上设计有不同的套餐，有免费套餐，有付费套餐，积分攒够后可以兑换一周的付费套餐使用。
  - 注册后分片一个号码，类似QQ号，可以设置头像，昵称，关注排行榜上的人，也可以通过号码加好友，查看好友每日每周的学习进度，勋章。
  - 生成一段 app 介绍文档，介绍 app 的目标人群，适用场景，用户痛点，解决方案，市场前景，app 亮点等，目标是投资人。
  - 技术栈用nuxt v4, sqlite, nuxt ui，支持email注册，谷歌，微信扫码登录，微信内自动登录，PC，移动端自适应
