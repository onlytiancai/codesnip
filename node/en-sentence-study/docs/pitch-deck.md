# SpeakFlow - 英语口语练习平台

## 一、项目愿景

SpeakFlow 是一款基于 AI 技术的英语口语练习工具，帮助用户将"心里想说的话"准确、地道地用英文表达出来。我们相信，语言学习的核心不是背单词，而是**表达思维的转变**。

## 四、用户痛点

### 痛点 1: 心里想得到，说不出来

> "我知道我想表达什么，但就是不知道怎么用英文说"

- 传统的英语学习侧重单词和语法，缺乏句型表达训练
- 用户能看懂英文，但无法主动输出
- 缺乏场景化的语境练习

### 痛点 2: 不知道自己的表达是否正确

> "我说了，但不知道说得对不对、地道不地道"

- 缺少即时反馈机制
- 语言交换对象难找、时间难以协调
- 没有专业点评，成本高（外教课 300-800元/小时）

### 痛点 3: 学习缺乏系统性和持续性

> "学了很久，但感觉进步不明显"

- 学习材料分散，没有体系化的练习路径
- 缺乏进度追踪和成就感
- 难以坚持，三天打鱼两天晒网

### 痛点 4: 不知道哪些表达是地道的

> "我的语法没问题，但老外听了觉得奇怪"

- 中式英语思维普遍存在
- 不知道口语和书面语的差异
- 不知道不同场景需要不同的表达风格

## 五、解决方案

### 核心功能

#### 1. 场景化句型练习

```
场景选择 → 任务选择 → 句子练习 → AI 点评

特点：
- 预置 8+ 实用场景，覆盖生活和工作高频场景
- 每个场景有 4-6 个任务，任务下有 6-10 个句子
- 句子来自真实场景，语言地道实用
- 进度可视化，完成度一目了然
```

#### 2. AI 智能点评系统

```
用户输入英文 → AI 分析 → 多维评分 + 改进建议

点评维度：
- 语法正确性（Grammar）
- 词汇选择（Vocabulary）
- 表达地道性（Naturalness）
- 场景适合度（Appropriateness）
- 礼貌性（Politeness）
- 综合评分（Overall 1-100）
```

**优势对比**:
| 对比项 | 传统方式 | SpeakFlow |
|--------|----------|-----------|
| 即时性 | 需要预约外教或语伴 | 24/7 即时响应 |
| 成本 | 300-800元/小时 | MVP 免费，远低于市场 |
| 专业性 | 取决于外教水平 | 统一标准，深度学习 |
| 隐私性 | 需要和陌生人对话 | 完全私密，无社交压力 |

#### 3. 提示系统

```
用户请求提示 → AI 生成 3-5 种候选表达

提示特点：
- 同一中文句子有多种英文表达
- 不同风格：正式、中性、口语化
- 说明每种表达的适用场景和语气
- 帮助用户扩展表达多样性
```

#### 4. 闪卡复习系统

```
学习过的句子 → 闪卡形式复习 → 标记学会

复习特点：
- 自动播放中英文，边听边记
- 可设置只听英文或中英对照
- 学会的句子不再重复出现
- 基于记忆曲线，科学复习
```

### 市场趋势

3. **碎片化学习**: 移动端使用场景增多，10-15分钟练习成为主流
4. **隐私意识**: 用户对开口说英语有心理障碍，私密练习需求强

### 竞争格局

| 竞品 | 类型 | 优势 | 不足 |
|------|------|------|------|
| 多邻国 | 工具型 | 用户多，品牌强 | 侧重单词，不够口语化 |
| 趴趴英语 | AI 口语 | 产品新颖 | 场景有限，点评深度不足 |
| 雅思哥 | 考试导向 | 考试群体精准 | 场景单一，非日常口语 |
| 口语侠 | 语伴匹配 | 真人互动 | 成本高，时间协调难 |

### 我们的机会

1. **场景化差异化**: 不是单词学习，是场景中的完整表达
2. **AI 深度点评**: 多维度专业点评，不是简单判断对错
3. **社交化运营**: 排行榜、邀请、勋章系统提升粘性
4. **轻量化 MVP**: 快速验证，快速迭代

## 七、App 亮点

### 亮点 1: 从"翻译思维"到"表达思维"

传统英语学习教的是"怎么翻译"，SpeakFlow 教的是"怎么表达"。
- 不是单词替换，是句型思维
- 不是语法正确，是场景适合
- 不是死记硬背，是情境记忆

### 亮点 2: AI 点评专业且即时

```
Grammar: 8/10 - 语法正确，但 "could you" 比 "can you" 更礼貌
Vocabulary: 7/10 - 用词准确，"receipt" 比 "bill" 更正式
Naturalness: 6/10 - 结构偏书面，口语中常说 "Yeah, sure!"
Appropriateness: 9/10 - 非常适合酒店场景
Politeness: 9/10 - 语气礼貌，得体
Overall: 78/100
Suggestions:
1. "I'd like to check in" 可以改为 "I've got a reservation under..."
2. 加入房间号能让句子更完整
```

### 亮点 3: 提示扩展表达多样性

```
你可以说：
1. "Could I get a window seat, please?" (正式，适合高档餐厅)
2. "Can I have a seat by the window?" (中性，日常可用)
3. "Window seat, please." (简洁，口语化)
4. "I'd prefer a window seat if that's possible." (委婉，有礼貌)
```

### 亮点 4: 社交化学习提升粘性

- **排行榜**: 每日、每周学习榜单，良性竞争
- **邀请码**: 邀请朋友学习，双方获得奖励
- **勋章系统**: 成就激励，持续学习
- **好友系统**: 类似 QQ，好友之间互相看进度

### 亮点 5: 积分制商业模式

```
免费用户: 每日 5 句免费练习
付费套餐: 每月 29 元，解锁无限练习
积分兑换: 攒积分换付费套餐（50积分 = 1周）
```

**优势**: 降低付费门槛，用户先用后付
