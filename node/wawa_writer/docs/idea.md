# AI 辅助写作工具 - 产品构思

## 产品定位

一边学习一边记录笔记的 AI 辅助写作工具

## 核心功能

### 1. 写作建议
- 输入标题后，AI 提供结构建议
- 输入标题后，AI 提供风格建议

### 2. 参考资料
- 输入标题后，搜索相关帖子/文章供参考学习

### 3. 内容管理
- 显示段落章节结构
- 支持段落级别的细化和优化

### 4. 内容增强
- 为段落配图

### 5. AI 评审
- 风格评审
- 内容评审
- 结构评审

## 产品决策

| 维度 | 选择 |
|------|------|
| 目标用户 | 通用/多场景（技术博客、营销文案、学术论文、创意写作等） |
| 搜索来源 | 搜索引擎 API + 特定平台 API + RSS 订阅源 + 本地知识库 |
| 配图方案 | AI 生图 + 图库搜索 + 截图/图表生成 + 手动上传（全支持） |
| 写作模式 | 对话式协作（通过对话逐步构建内容） |
| 输出格式 | Markdown、PDF、HTML/在线发布 |
| 技术栈 | 主流 Web 技术栈（React/Vue + Node.js + PostgreSQL/MongoDB） |

## 用户旅程

1. 用户输入文章标题
2. AI 提供结构和风格建议
3. 系统搜索相关资料供参考
4. 用户通过对话方式与 AI 协作写作
5. 逐段细化、优化、配图
6. AI 进行风格、内容、结构评审
7. 导出为 Markdown/PDF/HTML
