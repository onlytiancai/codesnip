# Wawa English Speaker

这是一个功能丰富的英语阅读器应用，使用 Vue 3 和浏览器 Speech Synthesis API 开发，提供以下核心功能：

## 主要功能

- **文本分析与分词**：将输入文本进行智能分词，保留原文标点与特殊符号。
- **离线音标获取**：通过本地 CSV 文件 `config/offlineIPA.csv` 为单词提供音标（IPA）支持。
- **语音朗读**：支持逐词朗读和逐句朗读，并在朗读时高亮当前单词或句子。
- **单词信息查询**：点击单词可查看详细信息，包括翻译、定义和变化形式。
- **句子翻译**：使用 Ollama API 将英文句子翻译成中文，支持流式翻译和自定义翻译提示。
- **设置调整**：可配置翻译选项、语音速度（rate）、音调（pitch）和音标显示。
- **键盘快捷键**：支持方向键切换句子，空格键播放/暂停朗读。

## 技术特点

- **纯浏览器运行**：无需服务器环境，直接在浏览器中运行。
- **模块化结构**：组件化设计，代码结构清晰，易于维护。
- **响应式布局**：适配不同屏幕尺寸，提供良好的移动端体验。
- **可配置性**：支持自定义 Ollama API 地址、模型名称和翻译提示。
- **离线支持**：音标查询完全离线，翻译功能需要 Ollama 服务支持。

## 使用方法

1. 在文本框中输入英文文本。
2. 点击"分析"按钮处理文本。
3. 使用悬浮控制栏或键盘快捷键进行朗读控制。
4. 点击单词查看详细信息。
5. 在设置面板中配置翻译和语音参数。

## 翻译功能配置

翻译功能需要本地运行 Ollama 服务：

1. 安装 Ollama（参考：https://ollama.com/）
2. 启动 Ollama 服务
3. 在应用设置中配置正确的 API 地址和模型名称

默认配置使用 `gemma3:4b` 模型，API 地址为 `http://localhost:11434/api/generate`。
