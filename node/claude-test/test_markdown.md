# 测试文档

这是一个测试 Markdown 文档，用于验证翻译功能。

## 功能列表

以下是一些主要功能：

1. **标题翻译**：保持标题层级
2. **段落翻译**：自动合并连续段落
3. **列表翻译**：支持有序和无序列表
4. **引用翻译**：保持引用格式
5. **代码保护**：代码块不会被翻译

### 使用说明

请按照以下步骤操作：

- 第一步：安装依赖
- 第二步：配置 API
- 第三步：运行翻译

> 注意：翻译时会保留原文的格式结构。

## 代码示例

```python
def hello_world():
    print("Hello, World!")
    return "success"
```
