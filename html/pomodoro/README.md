# 番茄钟应用

一个简单的番茄钟应用，帮助你提高工作效率和时间管理。

## 功能

- 三种工作模式：
  - 工作模式（25分钟）
  - 短休息（5分钟）
  - 长休息（15分钟）

- 计时功能：
  - 显示剩余时间
  - 开始/暂停/重置按钮
  - 自动在工作和休息模式之间切换
  - 每完成4个番茄钟后自动进入长休息

- 任务管理：
  - 添加任务
  - 标记任务完成
  - 删除任务

- 其他功能：
  - 计时结束后播放提示音
  - 显示已完成的番茄钟数量
  - 响应式设计，适配不同屏幕尺寸

## 如何运行

1. 确保已安装 Node.js
2. 在项目目录下运行：
   ```
   npm start
   ```
3. 在浏览器中访问：http://localhost:3000

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
- Node.js (用于本地服务器)
