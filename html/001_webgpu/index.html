浏览器里的 GPU 编程

四个 WebGPU 互动 demo，从噪声地形、GPU 粒子，到实时光线追踪、分形缩放。鼠标拖动、点击、滚轮，都能看到 GPU 在工作。

Vertex Shader · 噪声

01 · 3D 噪声波浪地形

256×256 顶点网格，4 层 FBM 噪声做基础地形。鼠标点击注入涟漪，拖拽旋转视角，滚轮缩放。颜色随高度从蓝到红渐变。

打开 demo
Compute Shader · 流场

02 · GPU 粒子星系

6 万个粒子在 curl noise 流场中演化。鼠标吸引、点击爆裂、1/2/3 切换模式（漩涡 / 星云 / 爆炸）。

打开 demo
Fragment Shader · Path Tracing

03 · 实时光线追踪球体

多个 3D 球体的金属/玻璃/漫反射材质，最多 4 次反弹。拖拽旋转视角，滚轮缩放，双击替换球体。

打开 demo
Fragment Shader · 迭代

04 · Mandelbrot · Julia 分形

平滑着色的分形图，鼠标点击/拖动平移缩放。按住 J 切到 Julia 模式，R 重置视图。

打开 demo
