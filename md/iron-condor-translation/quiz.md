## 引言 / Introduction

铁鹰（Iron Condor）由 4 个期权腿组成，本质上是牛市认沽价差 + 熊市认购价差的组合，在标的横盘时获取时间价值收益。

::: quiz q-intro-1 single
id: q-intro-1
section: intro
difficulty: easy
tags: definition, structure
question: |
  铁鹰（Iron Condor）本质上是哪两种价差的组合？
options:
  - { key: A, text: "牛市认购价差 + 熊市认沽价差" }
  - { key: B, text: "牛市认沽价差 + 熊市认购价差" }
  - { key: C, text: "跨式（Straddle）+ 宽跨式（Strangle）" }
  - { key: D, text: "蝶式（Butterfly）+ 日历价差（Calendar）" }
answer: B
explanation: |
  **为什么选 B**：铁鹰由"卖低 Put + 买更低 Put"的牛市认沽价差（Bull Put Spread），
  与"卖低 Call + 买更高 Call"的熊市认购价差（Bear Call Spread）组合而成。
  两边都**净收权利金**，都看好标的在到期前**停留在某个区间**内。
  实战意义：理解这个结构才能看懂后面的损益曲线、希腊字母、调整手法。

  **为什么其他选项不对**：
  - A：**方向写反了**。牛市认购价差是买低 Call + 卖高 Call（看多），
    熊市认沽价差是买高 Put + 卖低 Put（看空），两者都是净付权利金的 debot spread，
    不构成收权利金的铁鹰。
  - C：跨式（Straddle：买 ATM Call + 买 ATM Put）和宽跨式（Strangle：买 OTM Call + 买 OTM Put）
    都是**两腿做多波动率**策略，净付权利金，方向上与铁鹰"收权利金做空波动率"完全相反。
  - D：蝶式是 4 行权价但中间对称结构，日历价差是同行权价不同到期月——
    都不是铁鹰的组成。
:::

tags: abc-example, profit
question: |
  假设你在 ABC 股票（当前 $47.50）上建了一笔 **1 张**铁鹰，到期 10 月，
  4 个行权价为 $45/$50/$55/$60（卖 $50 Put / 买 $45 Put / 卖 $55 Call / 买 $60 Call）。
  两边价差的净权利金都是 **$1/股**。
  若到期日 ABC 收盘价落在 $50 与 $55 之间，账户最大总收益是？
options:
  - { key: A, text: "$100" }
  - { key: B, text: "$200" }
  - { key: C, text: "$300" }
  - { key: D, text: "$500" }
answer: B
explanation: |
  **为什么选 B**：**$1/股 × 100 股/张 = $100/张（每边）**，
  两边（Put 价差 + Call 价差）合计 **$200**。

  拆开看：
  - **每股 → 每张**：权利金按"每股"报价，1 张合约 = 100 股，所以 × 100
  - **两边相加**：Put 价差贡献 $100 + Call 价差贡献 $100 = $200
  - 不需要乘合约张数（因为这里只有 1 张）

  **实战提醒**：最大收益是"**理想状态**"——必须 4 个行权价都未触及，
  且持有到期。实战中通常在吃到 **50% 最大收益**时就主动平仓（见 NFLX 案例）。

  **为什么其他选项不对**：
  - A：只算了**一边**价差，漏了另一边 $100。
  - C：这是该铁鹰的**最大亏损**（$500 风险资本 − $200 最大收益）。
  - D：这是**风险资本**（$5 行权价差 × 100 股），与最大收益无关。

  **数量级提醒**：如果建仓是 **4 张合约**，最大收益就是 $200 × 4 = **$800**。
  注意"张数"在权利金计算里是乘出来的，不是加出来的。
:::

## 熊市认购价差 / Bear Call Spread

卖出低行权价 Call + 买入更高行权价 Call，构成中性偏空策略。权利金为净收入。

::: quiz q-bc-1 single
id: q-bc-1
section: bear-call
difficulty: easy
tags: structure
question: |
  熊市认购价差（Bear Call Spread）的结构是？
options:
  - { key: A, text: "买低 Call + 卖高 Call" }
  - { key: B, text: "卖低 Call + 买高 Call" }
  - { key: C, text: "买 Put + 卖 Put" }
  - { key: D, text: "买 Call + 买 Put" }
answer: B
explanation: |
  **为什么选 B**：熊市认购价差 = **卖低行权价的 Call + 买更高行权价的 Call**（同一到期月）。
  卖腿收权利金，买腿用来**封顶亏损**（最大亏损 = 行权价差 − 净权利金）。
  名字虽然带"熊市"，实质是**中性偏空**：赌标的到期前**不涨破**短腿行权价。

  **为什么其他选项不对**：
  - A：**完全反了**——这是牛市认购价差（Bull Call Spread），净付权利金做多。
  - C：买 Put + 卖 Put 是 Put 价差；如果是"买高 Put + 卖低 Put"才是熊市认沽价差。
  - D：买 Call + 买 Put 是**跨式/宽跨式**，做多波动率，与价差结构无关。

  **实战意义**：铁鹰的 Call 端用的就是这种结构。要分清"Bear Call = 中性偏空，
  赌不涨破"——不要被名字里的"Bear"误导成看大跌。
:::

question: |
  你建了一笔熊市认购价差：卖 10 月到期、行权价 $50 的 Call（收 $2），
  买 10 月到期、行权价 $55 的 Call（付 $1）。每张合约净收到的权利金是？
options:
  - { key: A, text: "$50" }
  - { key: B, text: "$100" }
  - { key: C, text: "$200" }
  - { key: D, text: "$300" }
answer: B
explanation: |
  **为什么选 B**：净权利金 = ($2 卖 − $1 买) × 100 股/合约 = **$100**。
  每股净收 $1，乘以合约乘数 100 即可。
  对应的最大亏损 = ($5 行权价差 − $1 净权利金) × 100 = **$400**。
  回报率 = $100 / $400 = **25%**（按风险资本计）。

  **为什么其他选项不对**：
  - A：少乘了 100（漏掉合约乘数）。
  - C：把单边价差当成了完整铁鹰的收益。
  - D：混淆了权利金与最大亏损。

  **实战意义**：铁鹰的 Call 端和 Put 端是**对称结构**，都会贡献约一半的权利金。
  理解单边价差的"每股 → 每张合约"换算（× 100）是基本功。
:::

## 组合：完整的铁鹰 / Putting It All Together

两笔价差合并后，最大风险资本略少、收益翻倍。

::: quiz q-combine-1 single
id: q-combine-1
section: combine
difficulty: easy
tags: payoff
question: |
  你在 ABC（当前 $47.50）上同时建了一笔牛市认沽价差（卖 $50 Put 收 $2 / 买 $45 Put 付 $1）
  和一笔熊市认购价差（卖 $55 Call 收 $2 / 买 $60 Call 付 $1），两边都是 **1 张合约**、10 月到期。
  把两边合成一笔铁鹰，**最大亏损**是？
options:
  - { key: A, text: "$200" }
  - { key: B, text: "$300" }
  - { key: C, text: "$400" }
  - { key: D, text: "$700" }
answer: B
explanation: |
  **为什么选 B**：先理解"最大亏损"在铁鹰里的真正含义。

  **关键概念：铁鹰的两边不会同时达到最大亏损**
  标的同一时刻只能在一个价格上——要么跌破 Put 短腿、要么涨破 Call 短腿，
  **不可能两边同时被击穿**。所以"铁鹰最大亏损"指的是**某一边的最大亏损**（通常取较大者）。

  计算（按 1 张合约）：
  - 单边最大亏损 = 行权价差 $5 − 净权利金 $1 = $4/股
  - 单边最大亏损（金额）= $4/股 × 100 股 = **$400/边**
  - 因为只可能一边被击穿，**铁鹰最大亏损就是 $400/边 − $1/股净权利金 × 100 股 = $300**

  简单记忆法：**风险资本 $500（= $5 行权价差 × 100）− 净权利金 $200 = $300**。
  这里的"风险资本"是**风险上限**（如果股价穿越所有行权价），而"净最大亏损"是**实际可能亏损**。

  **实战意义**：铁鹰的**最大亏损 > 最大收益**——这是后面"止损、调整"的根本原因。
  实战中**通过调整（滚动下移、加倍建仓）**通常能避免吃满这个最大亏损。

  **为什么其他选项不对**：
  - A：是**最大收益**（$100/边 × 2 边 = $200），不是亏损。
  - C：单边最大亏损**未扣除**净权利金（$5 × 100 = $500 但减去 $200 = $300）。
    实际答案应扣除已收权利金。
  - D：没有任何合理算法能算出 $700，混淆了多个概念。
:::

  沿用上题铁鹰（最大收益 $200、最大亏损 $300），这笔铁鹰的**潜在回报率**约为？

  - { key: B, text: "40%" }
  - { key: C, text: "66.67%" }
  - { key: D, text: "100%" }

explanation: |
  **为什么选 C**：回报率 = 最大收益 / 最大亏损 = $200 / $300 ≈ **66.67%**。
  即"在最理想的情况下，1 块钱的风险资本能换 0.67 块钱的利润"。

  **实战提醒**：
  - 这个 66.67% 是**理想状态的回报率**——前提是**持有到期且标的未触及任一行权价**。
  - 实际回报要打折扣：调整失败的成本、提前平仓的时间价值损耗、爆仓的尾部风险等。
  - 长期复利视角下，铁鹰实战回报通常远低于这个理论值。

  **为什么其他选项不对**：
  - A：20% 是**单边价差**的回报率（$100 收益 / $400 风险资本），
    不是合成铁鹰的。
  - B：没有合理算法对应 40%。
  - D：100% 意味着收益 = 亏损，对铁鹰永远不可能（最大亏损永远 > 最大收益）。
:::

## 何时建仓铁鹰 / When to Enter Iron Condors

建仓时机的 4 种场景：区间扩张之后、波动率飙升时、IV Rank 较高、随时建仓保持一致。

<!-- ### 区间扩张之后 / After a Range Expansion -->

::: quiz q-timing-1 single
id: q-timing-1
section: timing
difficulty: medium
tags: range-expansion
question: |
  区间扩张（Range Expansion）后建仓铁鹰的核心逻辑是？
options:
  - { key: A, text: "标的刚经历大涨或大跌，预期接下来进入震荡盘整" }
  - { key: B, text: "标的刚创新低，应该抄底做多" }
  - { key: C, text: "波动率在历史最低点，是建仓的最佳时点" }
  - { key: D, text: "VIX 跌破 10，说明市场极度贪婪" }
answer: A
explanation: |
  **为什么选 A**：铁鹰是"赌标的停在区间内"——所以最佳建仓时机是**预期盘整**。
  区间扩张刚结束时（布林带开口放大、波动率飙升后），市场通常会进入一段**收缩盘整**。
  配合较高的 IV（扩张带来的），是铁鹰的"**黄金窗口**"。

  **经典案例**：2016 年 11 月美国大选后 RUT 暴跌，布林带开口到近年最大，
  随后**整整 8 个月**（2016-11 到 2017-07）RUT 都在 1350-1450 区间内盘整——
  那是铁鹰历史上最赚钱的时期之一。

  **为什么其他选项不对**：
  - B："抄底"是股票思维，不是铁鹰思维。铁鹰赚的是时间价值，不是方向性涨幅。
  - C：IV 在**历史最低**时建仓 = 权利金很便宜、回归概率小，**反而是糟糕时机**。
  - D：VIX 跌破 10 意味着市场极度平静，权利金稀薄，建仓意义不大。
:::

<!-- ### 波动率飙升时 / On a Volatility Spike -->

::: quiz q-timing-2 single
id: q-timing-2
section: timing
difficulty: medium
tags: vol-spike, vega
question: |
  遇到市场大跌、VIX 飙升时，铁鹰交易员的最佳做法是？
options:
  - { key: A, text: "立刻重仓做空波动率" }
  - { key: B, text: "等上一两周再行动，避开 V 形反转" }
  - { key: C, text: "立刻买入跨式做多波动率" }
  - { key: D, text: "完全不做，等上一年再说" }
answer: B
explanation: |
  **为什么选 B**：铁鹰的本质动作是**卖期权（short vega）**——赚的是卖出期权的权利金，
  承担的是"卖贵之后还要应对 IV 继续走高"的风险。
  闪崩刚发生时，IV 虽然已经飙升，但**还可能继续走高**。此时建仓，
  后续 IV 若二次冲顶，**卖方头寸会因 vega 持续亏损**。
  等 1-2 周让恐慌尘埃落定，IV 通常从恐慌高位回落，但仍处于较高水平——
  这是"**IV 仍然够高（权利金不薄）+ 不再失控（继续冲顶概率小）**"的平衡点。

  **辅助理解：VIX 期货期限结构**：
  - 恐慌期：近月 IV 暴涨 → 期货曲线呈**贴水**（近高远低）。
  - 恐慌过后：曲线从贴水**回归升水**（近低远高）。
  这正是"市场情绪从失控回归理性"在期限结构上的反映。

  **为什么其他选项不对**：
  - A：VIX 飙升**后立刻重仓做空** = 在最可能二次冲顶时满仓 vega 暴露，
    容易在反弹前被打爆。
  - C：买入跨式是**做多波动率**，与铁鹰的"卖期权收权利金"方向相反。
    想对冲应该买 VIX Call 或更虚值 Put，而不是反手做跨式。
  - D："等一年"过度保守——闪崩后 IV 仍高，是铁鹰的优良窗口，
    完全错过可惜。关键是**等 1-2 周**而不是完全放弃。
:::

<!-- ### 根据 IV Rank / Based on IV Rank -->

::: quiz q-timing-3 single
id: q-timing-3
section: timing
difficulty: easy
tags: iv-rank
question: |
  关于 **IV Rank**（隐含波动率排名）这个指标，下列哪个理解是正确的？
options:
  - { key: A, text: "等于 VIX 指数的数值（如 18、25）" }
  - { key: B, text: "标的过去一年 IV 的算术平均值" }
  - { key: C, text: "当前 IV 在过去一年 IV 极值范围中的相对位置（0~100）" }
  - { key: D, text: "对未来 30 天实际波动的预测值" }
answer: C
explanation: |
  **为什么选 C**：IV Rank 的本质是**百分位排名**——
  把当前 IV 放在过去一年 IV 的最低值到最高值之间做线性映射，
  得到一个 0-100 的数字。公式：

  ```
  IV Rank = (当前 IV − 过去 1 年最低 IV) / (过去 1 年最高 IV − 过去 1 年最低 IV) × 100
  ```

  - IV Rank = 0：当前 IV = 过去 1 年最低
  - IV Rank = 100：当前 IV = 过去 1 年最高
  - IV Rank = 50：当前 IV 处于过去 1 年**正中间**

  **实战意义**：IV Rank 是铁鹰选标的的核心筛子。
  IV Rank 越高（如 > 50），说明期权**相对昂贵**，卖方赚的权利金越多；
  IV Rank 越低（如 < 20），权利金稀薄，铁鹰的"性价比"差。

  **为什么其他选项不对**：
  - A：VIX 是 VIX 指数本身的数值（如 18），与 IV Rank 是两个完全不同的指标。
  - B：IV 的算术平均值是**绝对水平**（如 25%），不是排名，
    无法反映"现在处于历史什么位置"。
  - D：预测未来波动的是**隐含波动率 IV 本身**或 GARCH 模型，
    不是 IV Rank（IV Rank 只反映**相对位置**）。
:::

  - D：LEAPS（>1 年）虽然 Gamma 更低，但 Theta 衰减极慢、资金占用大、收益曲线过于平坦，
    实操中并非作者首选。

  **实战意义**：长期铁鹰并非"完美策略"，但**对业余交易员更友好**——
  不需要每天盯盘，临场决策压力小。

## 理解升水与贴水 / Understanding Contango and Backwardation

VIX 期货的升水（Contango）和贴水（Backwardation）反映市场对未来波动的预期。

::: quiz q-cont-1 single
id: q-cont-1
section: contango
difficulty: easy
tags: definition
question: |
  **升水（Contango）** 状态下 VIX 期货期限结构的形态是？
options:
  - { key: A, text: "近月高于远月" }
  - { key: B, text: "近月低于远月" }
  - { key: C, text: "完全水平" }
  - { key: D, text: "随机波动" }
answer: B
explanation: |
  **为什么选 B**：Contango = **近月低、远月高**的期货曲线形态。

  **为什么是这样的曲线？**
  距离现在越远，发生波动事件的概率越大（"时间 = 不确定性"）。
  所以**远月合约需要更高的 IV** 来补偿持有风险，曲线自然向上倾斜。

  **实战意义**：
  - Contango 是市场**常态**——尤其在牛市/平静市。
  - 做空 VIX 的产品（如 XIV、SVXY）依赖**期货曲线从远月向近月回归**（contango roll yield）。
  - 对铁鹰交易员：Contango 状态 = 市场平静 = **正常建仓窗口**。

  **为什么其他选项不对**：
  - A：**近高远低** = 贴水（Backwardation），是恐慌形态，不是 Contango。
  - C：完全水平意味着远月和近月 IV 一样，时间价值被市场"无视"——理论上不存在。
  - D：期限结构是宏观层面的曲线形态，不会"随机波动"。
:::

## 分腿建仓 / Legging In to an Iron Condor

分腿建仓可提高收益但单边裸奔风险显著。

::: quiz q-leg-1 single
id: q-leg-1
section: legging
difficulty: easy
tags: definition
question: |
  "分腿建仓"（Legging In）指的是？
options:
  - { key: A, text: "4 个腿全部一次性下单" }
  - { key: B, text: "分多次下单，先建一侧等股价配合再建另一侧" }
  - { key: C, text: "用限价单分批平仓" }
  - { key: D, text: "用市价单快速建仓" }
answer: B
explanation: |
  **为什么选 B**：分腿建仓 = 把 4 条腿**拆开**，先建一侧价差（如 Bull Put），
  等股价移动（最好向有利方向）后再建另一侧（Bear Call）。
  目的是**追求更优的成交价**（等行情配合）。

  **实战意义**：
  - **收益**：抓到好的入场点位，**总权利金可能更高**。
  - **代价**：在两次下单之间，仓位处于"**单边裸奔**"状态——
    一侧承压时，另一侧没有任何对冲收益（参见 q-leg-2）。

  **进阶提示**：作者本人建仓时是"**先等一侧成交，迅速让另一侧成交**"——
  这与"分腿建仓"略有不同：**分腿 = 等行情配合后再建另一侧**（有间隔），
  而作者的方式是**尽量快速建完**（但仍分次）。

  **为什么其他选项不对**：
  - A：4 个腿一次性下单叫"**组合建仓**"，不是分腿。
  - C：是**分批平仓**（参见 q-timing-4 的 25% 阶梯出场），不是建仓。
  - D：用市价单"快速建仓"与分腿无关——分腿是**多次下单**，单次可以是限价单或市价单。
:::

## 选择行权价 / Selecting Iron Condor Strikes

3 种选价方法：Delta 法、标准差法、技术分析。

::: quiz q-strike-1 single

question: |
  作者推荐用 **Delta 法** 选铁鹰短腿（卖出腿）行权价时，短腿的 Delta 大约是？
options:
  - { key: A, text: "5 Delta" }
  - { key: B, text: "10-15 Delta" }
  - { key: C, text: "30 Delta" }
  - { key: D, text: "50 Delta" }
answer: B
explanation: |
  **为什么选 B**：作者本人选铁鹰短腿时，**Delta 通常放在 10-15** 这个范围。

  **Delta 的含义**：Delta ≈ 期权到期成为 ITM 的概率（粗略近似）。
  - 10 Delta = 大约 10% 概率成为 ITM
  - 15 Delta = 大约 15% 概率成为 ITM
  - 10-15 Delta 对应大约**1 个标准差外**的位置。

  **为什么是这个区间？**
  - Delta 越低（如 5）→ 越虚值 → 胜率越高，但单笔权利金很薄。
  - Delta 越高（如 30）→ 越接近平值 → 单笔权利金丰厚，但胜率显著下降。
  - **10-15 Delta 是"胜率 vs 收益"的甜点**：大约 **85-90% 的胜率**，权利金又不至于稀薄。

  **实战意义**：选 Delta 时要根据自己的**风险偏好**微调——
  保守派用 10 Delta（更高胜率），激进派用 15 Delta（更多权利金）。
  **关键是选完后保持一致，不要每次都改**。

  **为什么其他选项不对**：
  - A：5 Delta 虽胜率更高，但权利金太薄，性价比差。
  - C：30 Delta 胜率降到 70%，对铁鹰这种"多次小赢"的策略太冒险。
  - D：50 Delta ≈ 平值，胜率不到 50%——铁鹰完全不应该卖这种期权。
:::

  - 这次卖 15 Delta、下次改成 10 Delta、再下次凭感觉？
    → 心态会乱、回测无法做、长期胜率无法统计。
  - 选定一种方法，**死磕到底**，长期才有统计意义。

  **为什么其他选项不对**：
  - B：P/E、PEG、DCF 是**股票估值方法**，与期权行权价无关。
  - C：线性回归、移动平均、布林带是**技术指标**，不是"选行权价的方法"——
    布林带属于技术分析的一种工具。
  - D：随机/激进/保守是**风险偏好风格**，不是具体的行权价选择方法。

## Delta Dollars：最重要的指标 / Delta Dollars May Be the Single Most Important Metric

Delta Dollars = 持仓 Delta × 标的价格，反映方向性美元敞口。

::: quiz q-dd-1 single
id: q-dd-1
section: delta-dollars
difficulty: easy
tags: formula
question: |
  **Delta Dollars** 的计算公式是？
options:
  - { key: A, text: "Delta × 标的当前价格 × 100" }
  - { key: B, text: "持仓 Delta × 标的当前价格" }
  - { key: C, text: "Delta × 行权价" }
  - { key: D, text: "Gamma × 100" }
answer: B
explanation: |
  **为什么选 B**：Delta Dollars = **持仓 Delta × 标的当前价格**。

  **但要注意"持仓 Delta"指的是 share-equivalent delta**（股票等价 Delta），
  已经是 × 100 之后的数字。

  **它衡量什么？**
  - 单张合约的 Delta = 0.40（per-contract delta）
  - **Share-equivalent delta = 0.40 × 100 股 = 40 股**（等价于持有 40 股正股）
  - 股价 $100 → Delta Dollars = **40 × $100 = $4,000**
  - 这意味着**这份期权头寸的方向性敞口，相当于在股票上投入了 $4,000**

  **Delta 数值的两种常见写法**：
  | 写法 | 含义 | 例子 |
  | --- | --- | --- |
  | **小数（per-contract delta）** | 单张合约的 delta | 0.10、0.15、0.40 |
  | **整数 / 百分数（share-equivalent）** | 等价的股票股数 | 10、15、40 |

  两种写法本质相同，只是单位不同——
  **0.40 delta = 40 share-equivalent**。原文中"sell 10-15 delta"和"delta 0.40 = 40 shares"就是这个关系。

  **为什么要乘股价而不只是 Delta？**
  - Delta = 100 的头寸看起来很大，但如果标的是 $1 的股票，
    实际方向性敞口只有 **$100**——可以忽略。
  - Delta = 1 的头寸看起来很小，但如果标的是 $500 的股票，
    实际方向性敞口是 **$500**——比前一种情况大 5 倍。

  **所以 Delta 数字本身说明不了什么，必须 × 股价 才有意义**。

  **实战意义**：Delta Dollars 反映**账户整体的方向性美元敞口**——
  对比账户净值，就能判断"**方向性风险是否过大**"（参见 q-dd-2、q-dd-3）。

  **为什么 A 错（重要！）**：
  Delta Dollars 公式里**不需要再乘 100**。
  100 是合约乘数，已经在"share-equivalent delta"那一步用掉了。
  **0.40 × 100 = 40（share-equivalent）**，然后 **40 × $100 = $4,000**——
  不是 **0.40 × $100 = $40**（少了 × 100 这一步）。
  这一项混淆是常见错误。

  **为什么其他选项不对**：
  - C：行权价是固定数字，方向性敞口应该跟**当前股价**挂钩，不是行权价。
  - D：Gamma × 100 没有实际意义——Gamma 反映 Delta 的变化率，与"敞口金额"无关。
:::

## Delta 对冲 / Delta Hedging

紧急止血技巧：买 1 份 Put 把净 Delta 拉回中性。

::: quiz q-dh-1 single
id: q-dh-1
section: delta-hedge
difficulty: easy
tags: method
question: |
  你持有一笔铁鹰，随着市场**下跌**，净 Delta 已经变成 **+20**
  （原本中性偏空的方向性敞口被放大）。作者推荐的快速 Delta 对冲方法是？
options:
  - { key: A, text: "再卖一份 Put" }
  - { key: B, text: "买入约 20 Delta 的 Put 中和" }
  - { key: C, text: "用股指期货做空" }
  - { key: D, text: "完全平仓铁鹰" }
answer: B
explanation: |
  **为什么选 B**：买入一份 **20 Delta 的 Put**，把 +20 的净 Delta 中和回 0。
  这一份 Put 等价于"持有 -20 股正股"，正好抵消铁鹰的 +20 净 Delta。

  **操作思路**：
  - 净 Delta 偏正（+20）→ 市场跌 → 买负 Delta 的工具（Put）中和
  - 净 Delta 偏负（-20）→ 市场涨 → 买正 Delta 的工具（Call）中和
  - 简单口诀：**用什么 Delta 中和，就买对应方向的期权**

  **为什么这是应急工具，不是长期仓位？**
  - 它**只为短期止血**，给交易员**争取时间**思考要不要正式调整铁鹰。
  - Put 本身有时间价值损耗，**持有久了是亏的**。
  - 反弹时要及时退出（参见 q-dh-2）。

  **实战意义**：
  - Delta 对冲是**给铁鹰"穿救生衣"**，**不是替代铁鹰**。
  - 真正长期的方向性管理还是要靠**滚动下移 / 加倍建仓**等结构性调整。

  **为什么其他选项不对**：
  - A：再卖一份 Put 是**加仓卖期权**，方向与"想止血"相反，反而放大风险。
  - C：股指期货可以，但对小账户门槛太高、合约乘数过大，
    小幅对冲不灵活。**期权对冲**更精准。
  - D：完全平仓 = 放弃这笔铁鹰。如果只是暂时方向不对，没必要认亏出场；
    先对冲、等反弹再处理更合理。
:::

  **为什么选 B**：对冲 Put 的成本是**机会成本**（花时间价值买的"保险"）。
  一旦市场反弹、Put 的保护使命完成，**及时退出**。
  通常能**小亏**（花掉的时间价值）或**平价**（恰好回本）出场。

  - 对冲 Put 的本质是**保险**，风险事件过后保险就该退掉。
  - 继续持有 = 持续支付 Theta，每天都在亏时间价值。

  **实战教训**：
  - 业余交易员的常见错误是"**反弹后舍不得退出对冲**"，想着"还能再涨"，

  - 牢记：**对冲是工具，不是仓位**。

  **为什么其他选项不对**：
  - A：持有到期意味着 Put 可能在到期前归零（赚）也可能不行权（亏），
    但无论如何**白白浪费了 Theta**——专业做法是主动退出。
  - C：买入后立即卖出通常做不到获利（你刚买入，IV 还没充分体现下跌）。
  - D：滚动到下月 = 把短期对冲变成长期对冲，**成本激增**，
    不符合"应急工具"的定位。

## 理解 Gamma 风险 / Understanding Gamma Risk

Gamma = Delta 的变化率；临到期 + ATM 时最大。

::: quiz q-gamma-1 single
id: q-gamma-1
section: gamma
difficulty: easy
tags: definition
question: |
  Gamma（Delta 的变化率）在下列哪种情况下最大？
options:
  - { key: A, text: "深度虚值期权（Deep OTM）" }
  - { key: B, text: "平值期权（ATM）" }
  - { key: C, text: "深度实值期权（Deep ITM）" }
  - { key: D, text: "与 moneyness（行权价与现价关系）无关" }
answer: B
explanation: |
  **为什么选 B**：**平值（ATM）期权** 的 Gamma 最大。

  **直观理解**：
  - ATM 期权的 Delta 约等于 0.5——**对标的价格变动最敏感**。
  - 当股价小幅变动时，Delta 可能从 0.5 跳到 0.55 或 0.45（变化 0.05）。
  - 而深度虚值期权的 Delta 接近 0，**股价变动后 Delta 也几乎不变**。

  **为什么 ATM 的 Gamma 最大？**
  - Gamma 反映 Delta 曲线的**弯曲程度**。
  - ATM 期权的 Delta 曲线在 0.5 附近**最陡峭**（形如 S 形曲线的中段）。
  - 远离 ATM 后（深度 ITM 或 OTM），Delta 曲线**趋于平坦**（接近 0 或 1）。

  **Gamma 还跟什么有关？**
  - **到期时间**：临到期的 ATM 期权 Gamma 更大。
  - 这就是为什么"**Gamma Week**"（最后一周）让 short gamma 交易员胆战心惊。

  **实战意义**：
  - **铁鹰是 short gamma**——做空 Gamma 赚 Theta（时间价值），但要承担 Gamma 风险。
  - Gamma 大 = 股价小幅变动 → Delta 跳变 → 头寸剧烈波动。
  - 临到期 + ATM = **Gamma 最大 = 风险最高** = 必须避开或调整。

  **为什么其他选项不对**：
  - A：深度虚值 → Delta 接近 0、几乎不变 → Gamma 极小。
  - C：深度实值 → Delta 接近 1、几乎不变 → Gamma 极小。
  - D：Gamma 明显与 moneyness 相关，ATM 最大。
:::

  - C："前 1 周" = 刚建仓后的时间，Gamma 最低，最安全。
  - D：整个月 Gamma 风险不均匀，不能一刀切。

  - IV 低位 = 权利金便宜 + IV 继续下跌空间小 = **盈利概率低**

  所以应该**IV 高时加仓**（把握高胜率窗口）、**IV 低时降仓**（避免在差胜率时硬扛）。

  **底层逻辑**：这条原则不是机械的"追涨杀跌"，
  而是基于**波动率均值回归**这一统计规律——
  - IV 在历史极高位时，**未来回归的概率远大于继续走高**。
  - IV 在历史极低位时，**未来回归的概率远大于继续走低**。

  **动态仓位 vs 固定仓位**：
  - 固定 20% 仓位 = 不论 IV 高低都重仓同样多
  - 动态仓位 = IV Rank > 50 加仓到 25-30%、IV Rank < 20 降到 10-15%
  - 长期看**动态仓位收益曲线更平滑、最大回撤更小**。

  **实战经验**：
  - **IV Rank 是核心指标**（参见 q-timing-3）。
  - 仓位调整要**平滑过渡**，不要一天之内从 10% 跳到 30%。

  **为什么其他选项不对**：
  - A："放大收益"是结果，不是逻辑——风险也在放大。
  - B：股票收益与波动率的"正相关"在学术上有争议，**不是铁鹰策略的核心依据**。
  - D：低 IV 时**权利金便宜**是事实，但**性价比差**——
    因为盈利概率也低，并不是"低成本 = 好"。
:::

## 铁鹰实战案例 / Iron Condor Examples

两个案例：RUT 长期铁鹰 2018-01-30 建仓、NFLX 铁鹰 2018-07-18 建仓。

::: quiz q-ex-1 single
id: q-ex-1
section: examples
difficulty: medium
tags: rut-case, scenario
question: |
  假设你在 **2018-02-08** 持有一笔 RUT 长期铁鹰，
  RUT 在几天内从建仓的 $1575 跌到 $1508，Short Put Delta 已经触及 20。
  此时**最合理的调整动作**是？
options:
  - { key: A, text: "立即全部止损平仓" }
  - { key: B, text: "整体下移铁鹰（Put 端和 Call 端都下移到更接近现价的位置）" }
  - { key: C, text: "持有到期，不做任何动作" }
  - { key: D, text: "切换为跨式策略" }
answer: B
explanation: |
  **为什么选 B**：当 RUT 大跌导致 Short Put Delta = 20（远超 10-15 的初值），
  说明 Put 端**进入危险区**——继续跌可能吃满最大亏损。
  此时**整体下移**是最合理的结构性调整：
  - Put 端下移到更低行权价 → Put 端远离危险区 → 重新变成虚值
  - Call 端也下移 → 整体保持对称的 Delta 中性
  - 同时**多收一笔权利金**，补偿调整成本

  **作者 RUT 案例的真实路径**：
  - **2018-01-30 建仓**（RUT ~$1575）：3 手 90 日铁鹰
  - **2018-02-08 RUT 暴跌至 $1508**：整体下移（Put 端 1360-1340 → 1330-1310、Call 端 1760-1780 → 1700-1720）
  - **2018-02-14 继续跌**：加倍建仓，从 3 手加到 6 手
  - **2018-04-18 反弹**：平仓，**净赚 $1,075**

  **为什么这个调整有效？**
  - 下移 = 把"快要出问题的铁鹰"换成"还有充分缓冲的新铁鹰"。
  - 下移不是认亏，而是**给铁鹰续命**——把行权价拉回安全区。

  **为什么其他选项不对**：
  - A：RUT 暴跌但**还没吃满最大亏损**，立即止损 = 把可救的仓位砍了。
  - C：持有到期 = 看着铁鹰走入 Gamma 周，**风险爆炸式增长**（参见 q-gamma-2）。
  - D：切换跨式 = 从"卖期权"变成"买期权"，**完全改变策略方向**，
    不符合"调整"的范畴（调整是改进原策略，不是换策略）。

  **实战教训**：
  - 调整的关键是**"事件触发"**（Delta 触及 20）而非"时间触发"（每日检查）。
  - 这条规则要**事前写进交易计划**（参见 q-adj-1）。
:::

::: quiz q-ex-2 single
id: q-ex-2
section: examples
difficulty: medium
tags: nflx-case, scenario
question: |
  假设你在 **2018-07-18** 以 $376 建仓 NFLX 铁鹰（一周后 NFLX 跌到 $356），
  此时 Short Call 的 Delta 已经从初值跌到 **2**，
  这意味着 Call 端几乎失效（深度虚值、价值几乎归零）。
  此时**最合理的调整动作**是？
options:
  - { key: A, text: "立即全部止损平仓" }
  - { key: B, text: "把 Call 端下移到更接近现价的位置，重新赚取权利金" }
  - { key: C, text: "持有到期不动" }
  - { key: D, text: "全部平仓后再开新仓" }
answer: B
explanation: |
  **为什么选 B**：当 Short Call Delta 跌到 2，**意味着 Call 端几乎失效**：
  - 时间价值只剩 $0.14/股，对仓位几乎无贡献；
  - 但 Call 端的"风险封顶"作用也跟着消失——整个铁鹰变成了**纯 Put 价差**。
  - 此时下移 Call 端有两个好处：
    1. **多吃一笔权利金**（重新变成有价值的虚值 Call）
    2. **拉回 Delta 中性**（让铁鹰恢复对称结构）

  **作者 NFLX 案例的真实路径**：
  - **2018-07-18 建仓**（NFLX ~$376）：5 手铁鹰
  - **一周后 NFLX 跌到 $356**：Short Call Delta 跌到 2
  - **Call 端从 470-480 下移到 440-450**：多吃一笔权利金 + Delta 中性化
  - **2018-08-08 平仓**：吃到约 50% 的潜在最大收益，**净赚 $295**

  **这个调整的本质**：Call 端下移 = **用"赚更多钱"替代"被动等归零"**。
  即使 NFLX 横盘不动，Call 端下移后能**继续贡献 Theta**，整个仓位更健康。

  **为什么其他选项不对**：
  - A：NFLX 只是从 $376 跌到 $376 → $356（约 -5%），**没到止损线**，
    立即止损 = 把可调整的仓位砍了。
  - C：持有到期 = Call 端继续失效、Put 端压力不减，**铁鹰结构严重失衡**。
  - D：全部平仓再开新仓 = **额外的交易成本 + 时点风险**（NFLX 还在跌），
    远不如原仓位调整。

  **实战教训**：
  - "调整"不是"砍仓重开"，而是**改进原仓位**。
  - 当某一端失效时，**下移该端**是经典解法。
:::

  作者对**单笔铁鹰**的硬止损阈值是 **$1,000**（无论怎么想扛，触及即止损）。

options:

  - { key: C, text: "为了省税费" }
  - { key: D, text: "完全是凭感觉" }

explanation: |
  **为什么选 B**：$1,000 不是随意数字，而是"**事前定好的金额红线**"——
  - 把**单笔最大亏损**锁死在 $1,000
  - 即使**主观很想扛**也要止损，避免"**希望不是策略**"的陷阱

  **这条规则的实战意义**：
  - 作者在新冠崩盘中有**两笔交易被这条硬止损打掉**。
  - 事后看，那两笔如果死扛，**亏损会更惨烈**（甚至吃掉几个月的利润）。

  - 但作者本人铁鹰策略**仓位分散、单笔风险可控**，所以敢用**绝对金额**（$1,000）。
  - 关键是**红线不可动摇**——不是"差不多到 $1,000 就平"，而是"触及即平"。

  **为什么其他选项不对**：
  - A：账户规模与硬止损**没有直接关系**。$20,000 账户也能用 $1,000 止损（占 5%）。
  - C：硬止损与税务无关。
  - D："凭感觉"违反了**事前定规则**的核心原则（参见 q-adj-1）。

  **实战教训**：
  - 业余交易员最常见的爆仓原因：硬止损设了但**临场改规则**。
  - 一旦你"**这次再扛一下**"，硬止损就形同虚设。

## 结语 / Conclusion

经验无可替代；小资金起步盯风险；铁鹰是核心策略。

::: quiz q-con-1 single
id: q-con-1
section: conclusion
difficulty: easy
tags: principle
question: |
  文章结语中，作者对铁鹰交易员**最核心**的建议是？
options:
  - { key: A, text: "赚快钱、All-in 策略，最大化收益" }
  - { key: B, text: "小资金起步、严守风控，理论结合实战" }
  - { key: C, text: "跟着别人的单做，省心省力" }
  - { key: D, text: "每天都交易，保持高频活跃" }
answer: B
explanation: |
  **为什么选 B**：作者结语的核心信息是——
  "**Start with a small account ... learn risk management first**"
  （**小资金起步，优先学风控**）。

  **为什么这条建议最核心？**
  - **理论 ≠ 实盘**：期权策略的书能读完，但实盘中的**心理压力、临场决策、滑点**无法在书里学会。
  - **小资金试错**：小资金起步让你"**用真金白银换经验**"，但不会因为一次失误出局。
  - **风控优先**：风控能力**先于**盈利能力——活下来的人才能长期赚钱。

  **作者本人从 2004 年开始做期权，铁鹰至今仍是他的核心策略**——
  这不是"短期暴利"的故事，而是"**长期稳定复利**"的范例。

  **为什么其他选项不对**：
  - A："All-in"和"赚快钱"在前文被反复批评——
    100% 满仓 = 爆仓只是时间问题（参见 q-ret-1）。
  - C："跟着别人的单" = 完全放弃自己的判断力，
    一旦跟单对象失误你也不知道为什么。
  - D："每天都交易"违反一致性原则（参见 q-timing-4），
    还会因过度交易累积手续费。

  **实战意义**：
  - 读完文章后第一件事不是"开仓 100 张铁鹰"，
    而是**用 $2,000-$10,000 开一个真实账户**，
    严格按事前计划交易，记录每一笔。
  - 一年后再考虑加仓。

  **结语共鸣**：作者用 **"Trade safe!"**（交易顺利！）作为全文签名——
  "safe" 既是祝交易员安全，也是提醒"**风控优先**"。
:::
