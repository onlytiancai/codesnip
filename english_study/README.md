
### build.js

本脚本用于批量将当前目录中的 Markdown 文件（`.md`）转换为对应的 HTML 文件，并自动生成导航页面。其主要功能如下：

* **Markdown 转 HTML**：将每个 `.md` 文件转换成同名 `.html` 文件，并自动加入 UTF-8 编码声明，避免乱码。
* **增量构建**：记录并比较源文件与目标文件的修改时间，仅在 Markdown 文件更新后才重新生成对应的 HTML，未变更的文件会被自动跳过，提高构建效率。
* **自动生成首页 index.html**：扫描所有 Markdown 文件，按文件名字典序生成一个 HTML 索引页面，包含链接指向每个生成后的 HTML 页面。
* **时间戳同步**：生成 HTML 后自动将其修改时间同步为原始 Markdown 的修改时间，确保下一次执行时能正确判断是否需要更新。
* **无侵入、零配置**：不需要额外目录结构或配置文件，将脚本放在目标目录执行即可完成全部处理。
